Guidelines for Describing Bar Charts

Link to Full NCAM Guideline


Example: Figure 1 is a bar chart titled “Vaccination Coverage Among Children,” that measures the percentage of children vaccinated in four states over one year, from 2006 to 2007. Data were collected over three waves. In each state, the coverage increases over time. The data are summarized in the following table:

  Arizona Michigan Illinois Delawere
W1 35% 5% 0% 15%
W2 45% 33% 24% 27%
W3 50% 42% 41% 40%


Guidelines:

  • Provide type of chart, title, axis labels, and units of measurement
  • Briefly describe apparent trends in the data
  • Convert data into a table using column and row headings, or a list if appropriate
  • Ignore visual attributes that don’t contribute information necessary to answer a specific question
  • Do not include descriptions already provided in the surrounding text (e.g. if the key already alludes to three waves, this need to be repeated in the longer description)
